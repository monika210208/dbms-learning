📘 What is DBMS?

A DBMS is software that manages databases efficiently.

  • Data storage
  • Security
  • Data retrieval

📊 ER Model

Objects like Student, Course
Properties like Name, Age
Links between entities

📋 Relational Model

Data is stored in tables (relations).

  • Rows = Tuples
  • Columns = Attributes

📉 Normalization

No repeating groups
No partial dependency
No transitive dependency
